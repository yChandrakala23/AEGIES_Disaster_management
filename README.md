## Overview

**AEGIES** is an AI-powered disaster management platform designed to assist emergency monitoring, risk analysis, and response coordination during disaster situations. It combines real-time monitoring, AI-driven insights, and interactive visualization to help authorities and users make faster, smarter decisions.

🔗 **Live Application:** [https://aegies-disaster-management-fbgo2ph2a.vercel.app](https://aegies-disaster-management-fbgo2ph2a.vercel.app)

## Features

- 🗺️ **Real-time disaster monitoring dashboard**

## Installation & Local Setup

**Prerequisites:** Node.js

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure environment variables

Create a `.env.local` file in the root directory:

```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
